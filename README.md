# Binary Mask Maker

1. Add your dataset to the project root directory. 
2. In 'paint.py' set root_path = your dataset root directory
3. Identify the excel file for your dataset and set df = pd.read_excel("Path to excel file")
4. If starting for the first time delete conents of img_masks_link.json, and just leave a set of curly braces, i.e {}, also delete the contests of the 'masks' folder
5. Run paint.py
----------
### Other notes

Be sure to run paint.py with a python environemnt that has pandas, numpy and opencv-python
